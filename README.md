# 🎇 Python Diwali Sales Analysis

A complete **Exploratory Data Analysis (EDA)** project on Diwali sales data using Python. This project analyzes customer purchasing behavior during the Diwali festival season and provides valuable business insights through data cleaning, visualization, and statistical analysis.

---

## 📌 Project Overview

The objective of this project is to analyze Diwali sales data to understand:

- 🛍️ Customer purchasing patterns
- 👫 Gender-wise and age-wise sales trends
- 🗺️ State-wise and occupation-wise purchasing behavior
- 📦 Product category performance
- 💰 Revenue-generating customer segments

This analysis helps businesses improve **marketing strategies**, **customer targeting**, and **sales performance** during festive seasons.

---

## 📂 Dataset Information

The dataset contains customer transaction details such as:

| Column | Description |
|---|---|
| User ID | Unique customer identifier |
| Gender | Male / Female |
| Age Group | Age bracket of the customer |
| State | Customer's state |
| Occupation | Customer's profession |
| Product Category | Category of purchased product |
| Orders | Number of orders placed |
| Amount | Total amount spent |

**Dataset File:** `Diwali Sales Data.csv`

## 📊 Key Analysis Performed

### ✅ Data Cleaning
- Removed null/missing values
- Fixed incorrect data types
- Dropped irrelevant columns (`Status`, `unnamed1`)

### ✅ Exploratory Data Analysis (EDA)
- **Gender Analysis** – Sales distribution across Male and Female customers
- **Age Group Analysis** – Identifying the most active buying age groups
- **State-wise Sales Analysis** – Top revenue-contributing states
- **Occupation Analysis** – Purchasing power by profession
- **Product Category Analysis** – Best-performing product categories
- **Top-Selling Products** – Highest ordered individual products

### ✅ Data Visualization
- Bar Charts
- Count Plots
- Histograms
- Grouped Bar Plots

---

## 📈 Key Insights

> 💡 **Insight 1:** Married women aged **26–35 years** are the highest buyers during Diwali.

> 💡 **Insight 2:** Customers from **Uttar Pradesh, Maharashtra, and Karnataka** contribute the most to total sales.

> 💡 **Insight 3:** Professionals in **IT, Healthcare, and Aviation** sectors exhibit higher purchasing power.

> 💡 **Insight 4:** **Food, Clothing, and Electronics** are the top-performing product categories during the festive season.
